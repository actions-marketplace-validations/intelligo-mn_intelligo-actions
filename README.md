
# Intelligo Actions

Github action for Intelligo Framework. :rocket:

## Code in Master

Install the dependencies  
```bash
$ npm install
```

Build the typescript
```bash
$ npm run build
```

Run the tests :heavy_check_mark:  
```bash
$ npm test

...
```

## Usage:

```yaml
uses: intelligo-systems/typescript-action@v1
with:
  milliseconds: 1000
```
